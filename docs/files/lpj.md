# Launchpad project descriptor (LPJ)

This file format holds project metadata stored in a JSON format

## Format

### Version 1

```json
{
    "$schema": "../internal/schemas/project_v1.json",   //[ Schema file for Project V1 files. ]
                                                        //[ Needs to end in "project_v1.json" ]

    "title": "Project",                                 //[ Project title ]
}
```
